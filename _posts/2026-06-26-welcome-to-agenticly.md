This is the first post on **Agenticly** — a blog about building software with
AI agents.

The site is a Jekyll blog hosted on GitHub Pages and served at
`agenticly.app/blog` through a small Cloudflare Worker that proxies the path.
Posts live in `_posts/` as Markdown files named `YYYY-MM-DD-title.md`.

To add a new post, drop a Markdown file in `_posts/` with front matter like the
block at the top of this file, commit, and push to `main`. GitHub Pages rebuilds
the site automatically.
